## Waku

Waku is a simple tool to resize images to the aspect ratios you need for popular platforms like Instagram, Facebook, WhatsApp, and X. Quickly prep your images, preview them, and apply borders that match the destination.

### Features
- Resize to the right aspect ratio for target platforms.
- Add borders that match the selected aspect ratio.
- Customize border colors (defaults to white).
- Choose neutral, blurry, or gradient borders pulled from the image.

### Getting Started
```bash
bun install
bun run dev
```
Then open http://localhost:3000.
